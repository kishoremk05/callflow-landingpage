# CallFlow International

CallFlow International — Browser-based VoIP SaaS platform for international calling, video meetings, and enterprise communication.

## Tech Stack

- **React 18** + TypeScript
- **Vite** — Dev server & build tool
- **Tailwind CSS** — Utility-first styling
- **GSAP** — Scroll-triggered animations
- **Framer Motion** — Page transitions
- **Lenis** — Smooth scrolling

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

The dev server runs at `http://localhost:8080`.

## Project Structure

```
src/
├── components/    # Reusable UI components
├── pages/         # Route pages (Index, Features, Platform, Pricing)
├── lib/           # Data & utilities
├── hooks/         # Custom React hooks
└── assets/        # Static assets
```
